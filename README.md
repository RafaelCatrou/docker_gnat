## Goal

Provide GNAT GPL 2016 as a [Docker](https://www.docker.com/) container.

## Demo

### Get the image

```bash
sudo docker pull rafaelcatrou/docker_gnat
```

### Run

```bash
sudo docker run --rm -t -i rafaelcatrou/docker_gnat /bin/bash
```
